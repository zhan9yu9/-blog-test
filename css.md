# CSS知识总结

## CSS需要注意的地方

1、所有符号都是英文符号，区分大小写。最后必须以;结束。

2、不存在块级元素和内联元素，所有元素都可以是内联或者块级元素。其中inline元素中不能写border。

3、不要设置width=100%

4、写代码前必须要写border！！

5、Margin合并 只有上下重叠没有左右重叠。

6、css第一行写*{margin：0；padding：0；box-sizing：borderbox;}
